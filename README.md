# dotfiles

Prerequisite

- `rsync`

Install dotfiles

```sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/latentlect/.dotfiles/main/.utils/install.sh)"
```


Remove dotfiles

```sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/latentlect/.dotfiles/main/.utils/uninstall/uninstall.sh)"
```
